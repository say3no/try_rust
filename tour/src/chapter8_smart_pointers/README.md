## References Revisited

参照は基本的には、メモリ内のいくつかのバイトの開始位置を表す数値にすぎません。その唯一の目的は、特定の型のデータがどこに存在するかという概念を表現することです。**参照が単なる数値と異なるのは、参照の有効期限が参照したものよりも長くないことを Rust が検証することです。(そうしないと使用時にエラーが発生します)**
